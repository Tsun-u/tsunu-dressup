# 阿宇紙娃娃換裝間 🧸

一個純前端的 chibi 紙娃娃換裝小遊戲——幫阿宇穿衣服、戴頭飾、開關眼鏡，還能下載你的穿搭成果。

## 玩法

- **服裝**：可複選疊穿（含傳說中的完全體配置）
- **頭飾**：草莓、西瓜、青蛙、小狗、皇冠、派對帽
- **眼鏡**：可開關（預設戴著，畢竟是標配）
- **✨ 一鍵完全體**：你知道的
- **下載合成圖**：把目前穿搭存成 PNG

## 技術

單檔 `index.html`，純 HTML + CSS + JavaScript，無框架無後端。
部件為預先對位的透明 PNG 圖層，直接疊放渲染，`canvas` 合成下載。
